## Run Locally

Clone the project

```bash
  git clone https://github.com/deuriib/wasm-base-project.git
```

Go to the project directory

```bash
  cd wasm-base-project
```

Install dependencies

```bash
  dotnet restore
```

Start the server

```bash
  dotnet run
```
